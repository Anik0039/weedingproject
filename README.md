# Dream Intimacies - Wedding Photography Website

An interactive, 3D-themed wedding photography website built with Next.js 14.

## Features
- 🎨 **6 Love-themed Color Palettes** - Red Rose, Royal Plum, Soft Blush, Deep Wine, Lavender Love, Burgundy Gold
- ♥ **Love Transition Effects** - Hearts burst, rose petals, bokeh when navigating
- 🖼️ **3D Tilt Gallery** - Mouse-reactive perspective cards
- 📸 **Filterable Portfolio** - Wedding, Couple, Portrait categories
- 🎬 **Lightbox Viewer** - Full-screen photo viewing with navigation
- ✨ **Parallax Effects** - Floating rings, rose petals, mouse-tracking
- 📱 **Fully Responsive** - Mobile-friendly hamburger menu
- 🔄 **Auto-rotating Hero** - Ken Burns zoom slideshow
- ⏱️ **Interactive Timeline** - Animated wedding journey

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build for Production

```bash
npm run build
npm start
```

## Tech Stack
- Next.js 14 (App Router)
- React 18
- CSS-in-JS (inline styles)
- Google Fonts (Lora, Raleway, Great Vibes, Marcellus)

## Theme Switcher
Click the 🎨 button (bottom-right) to switch between 6 love-themed color palettes.
